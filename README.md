#  Brew Haven Cafe – Website Project  
## Task 03 – Full Stack Web Development Internship (FS Track)

---

##  Project Overview

Brew Haven Cafe is a multi-page responsive website developed as part of Task 03 under the Full Stack Web Development Internship (FS Track).

This project simulates a real-world cafe website where users can explore the menu, learn about the cafe, contact the team, add items to the cart, and submit feedback.

The goal of this task was to strengthen frontend development skills, improve UI structuring, and implement interactive features using JavaScript.

---

##  Key Features

-  Home Page with attractive landing section  
-  Menu Page displaying cafe items  
-  Cart functionality with dynamic item count  
-  Checkout page  
-  Contact page  
-  Feedback form with rating system  
-  Form validation  
-  Success page after submission  
-  Fully responsive design
